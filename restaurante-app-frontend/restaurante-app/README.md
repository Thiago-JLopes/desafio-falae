# Restaurante App - Frontend

Este é o frontend da aplicação Restaurante App, desenvolvido com React e TypeScript.

## Configuração

### Requisitos

- Node.js

### Passo a Passo

1. Instale as dependências:
   ```bash
   npm install
2. Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
## Tecnologias Utilizadas

- React
- TypeScript
- Tailwind CSS
- Vite
